Mondrian

XML Reference

Mondrian provides a sparsely-used XML namespace mondrian: to record metadata that is used to maintain important parts of the UI state between sessions.

This namespace is mutable; it is subject to grow and change as Mondrian is developed. However one of the design goals in this project is to minimize the "fluff" kept in SVG source files; you can expect this namespace to remain relatively bare.

Attributes

  • mondrian:angle

    Value: number [0-360]

    Example: mondrian:angle="5.21489"

    By default shapes remember which angle they have been rotated to so the transformer can remain relative to their original reference surface. Persisting their current angle with this attribute allows us to maintain this feature throughout many sessions with a single file.
